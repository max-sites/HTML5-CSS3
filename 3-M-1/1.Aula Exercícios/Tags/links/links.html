Links

Link interno

_self - vai abrir o link na janela ou frame atual next - Indica que o link é para próxima parte do documento atual

_top - vai desfazer todos os frames e abrir o destino no navegador completo prev - indica que o link é para a parte anterior do documento atual

_parent - similar ao uso do _top em uma referência à janela mãe nome-do-frame caso esteja usando frames, indica o nome da janela a abrir author - indica que é um link para o site do autor do artigo atual

Links externos em uma nova guia

_blank - abre perfil em nova Janela em branco external - indica que é um link para outro site que não faz parte do site.
