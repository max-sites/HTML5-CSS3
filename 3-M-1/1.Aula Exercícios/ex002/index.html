Parágrafos e quebras de linha.

Para criarmos um hipertexto precisamos colocar a tag <h1>(Hipertexto)</h1>.
Para criarmos uma linha horizontal precisamos colocar a tag <hr>.
Para fazermos a quebra de linha precisamos colocar a tag <br>.

Para criarmos um parágrafo, precisamos colocar a tag <p>(Parágrafo)</p>.

Para escrevermos o simbolo de MENOR e MAIOR precisamos digitar .
Para fazermos um comentário precisamos digitar <!--Comentário-->

Vamos adicionar alguns Símbolos especiais:

®, 𝕔, ™ €, £, ¥, ¢, ∅, ∑, Δ, ←, ↑, →, ↓, ☀, ♰.

Vamos adicionar alguns Emojis:

🖖, 🤓, 🕹, 🎮, 🤑, ♿, ⚓
